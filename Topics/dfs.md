# Depth-First Search

DFS traverses the *depth* of a graph before the width. It's pretty much synonymous with recursion.

The main components are a *base case* and an *inductive step*.

```
dfs(params) {
	if end of path:
		// perform whatever code
		return;

	dfs(diff_params)
}
```
